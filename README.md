# html

## 標籤語法

|語法|說明|
|:-:|-|
|<h1\>~\<h6\>|<h3>h3標題大小</h3>|
|\<p\>|段落|
|\<br\>|換行|
|\<hr\>|分割線<hr>|
|\<u\>|<u>底線</u>|
|\<b\>|<b>粗體</b>|
|\<a\>|超連結|
|\<img\>|圖片|
|\<ul\>|列表|
|\<table\>|表格|
|\<s\>|<s>刪除線</s>|
|\<mark\>|<mark>螢光標示</mark>|


### html特殊符號
|語法|說明|
|:-:|-|
|\&nbsp;|空格|

### \<a>標籤 => 屬性設定
- 超連結另開網頁
    - 在\<a>加入文字 target="_blank"
- 提示文字
    - 在\<a>加入文字 title="XXXXXXXX"
- href=""不帶網址時，代表「直接刷新本頁」

### \<img>標籤 => 屬性設定
- scr 放圖片網址
- alt 超連結的提示文字
- width 圖片大小
